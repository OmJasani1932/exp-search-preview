# Search Debugger

A React + TypeScript + Vite application for debugging a Search API across multiple scenarios.

## Tech Stack

- **React 18** — UI framework
- **TypeScript** — Type safety
- **Vite 4** — Fast build tool with HMR

## Getting Started

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm run preview
```
